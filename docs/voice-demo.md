# Voice Demo API

Spin up a capped, unauthenticated LiveKit voice session for embedding a public 'try the voice agent' widget.

> The voice demo lets visitors try a voice agent without an account. Sessions are **time-capped** and **quota-limited per browser fingerprint** to prevent abuse. For production voice agents use the [Voice Sessions API](/docs/voice-sessions-api) instead.
